CONVERT IHPGREENVILLE.COM TO A FULLY STATIC SITE — DEPLOY TO NETLIFY, ELIMINATE RAILWAY
You are converting the ihpgreenville.com Express/TypeScript server-side rendered site into a fully static site. The output is a /dist folder of plain HTML files that deploys to Netlify. After this is complete, Railway is no longer needed — no server, no database at runtime, no ongoing hosting cost.

UNDERSTAND WHAT YOU ARE BUILDING
A static site generator is a build script that runs once and produces HTML files. The script:

Reads all content (service data, condition data, blog posts) from source files
Runs the existing renderer functions against that content
Writes the output to /dist/[path]/index.html for every page
Generates sitemap.xml, robots.txt, llms.txt as real files in /dist
Copies all static assets (CSS, JS, images, fonts) to /dist

The resulting /dist folder contains everything the site needs. Netlify serves it directly with zero server involvement.

typescriptasync function writePage(urlPath: string, html: string) {
  // /services/acupuncture-therapy → dist/services/acupuncture-therapy/index.html
  // / → dist/index.html
  const filePath = urlPath === '/' 
    ? path.join(DIST, 'index.html')
    : path.join(DIST, urlPath.replace(/^\//, ''), 'index.html');
  await fs.ensureDir(path.dirname(filePath));
  await fs.writeFile(filePath, html, 'utf8');
  console.log(`✓ ${urlPath}`);
}
