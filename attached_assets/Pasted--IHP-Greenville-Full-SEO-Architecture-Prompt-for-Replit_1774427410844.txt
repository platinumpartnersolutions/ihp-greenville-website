## PART 1: TECHNICAL FOUNDATION
### Apply to every single page — no exceptions

**Server-Side Rendering Requirements:**
All critical content must be present in raw HTML source before any JavaScript executes. Google and LLM crawlers (ChatGPT, Perplexity, Claude, Gemini) must be able to read the following without executing JS:
- H1 tag
- First 2 paragraphs of body copy
- Phone number as a `tel:` link
- Navigation links and dropdown structure
- Meta title, meta description, canonical tag
- All schema markup as JSON-LD in `<head>`
- Sitemap link in robots.txt

**Performance Requirements:**
- All pages must load under 2.5 seconds on mobile LTE
- No render-blocking scripts above the fold
- Images: use WebP format, explicit width/height attributes, lazy loading on below-fold images, eager loading on hero image
- CSS: critical CSS inline in `<head>`, non-critical deferred
- No large JS bundles — vanilla JS only, no frameworks

**robots.txt:**
```
User-agent: *
Allow: /
Sitemap: https://www.ihpgreenville.com/sitemap.xml
```

**sitemap.xml:**
Generate a complete XML sitemap including every page in this document. Format:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://www.ihpgreenville.com/</loc>
    <priority>1.0</priority>
    <changefreq>weekly</changefreq>
  </url>
  <!-- all other pages at priority 0.8 -->
</urlset>
```

**llms.txt (place at /llms.txt):**
This file tells AI crawlers how to understand and cite the site. Include:
- Site name and description with Dr. Hendry's real differentiators (Prisma Health study, DAOM credential, 25 years experience, opioid alternative research)
- Every page URL with a 2–3 sentence description of what that page covers
- Conditions treated list
- Services list with brief descriptions
- Location and service area

**Canonical tags:** Every page must have `<link rel="canonical" href="https://www.ihpgreenville.com/[page-url]/">` in `<head>`.

**Open Graph tags (every page):**
```html
<meta property="og:title" content="[Page Title] | Integrative Health Partners">
<meta property="og:description" content="[Page meta description]">
<meta property="og:image" content="https://www.ihpgreenville.com/images/og-default.jpg">
<meta property="og:url" content="https://www.ihpgreenville.com/[page-url]/">
<meta property="og:type" content="website">
```
